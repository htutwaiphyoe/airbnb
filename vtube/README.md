# Vtube - Video Player

Powered by YouTube API

See in action [Vtube](https://vtube-292802.web.app)

## Technologies

- React Hooks
- Redux
- Redux-Thunk
- React-Redux
- React-Router-Dom
- Axios
- Material-UI React
- Firebase

## Features

- Most Popular Videos
- Search any video
- Play videos
- 50 videos only
- Skeleton Loader
- Responsive
- PWA
- Hosting
